Metrics Section:

  'm_all' – Run ALL metrics
  'm_ttr' – Run Time to Resolve
  'm_ddr' – Run Daily Discovery Rate
  'm_mdr' – Run Monthly Discovery Rate
  'm_ydr' – Run Yearly Discovery Rate
        Value is on or off – on will run the specific analysis, off will skip it.

Note:  true|false, 1|0 or yes|no will also work.

Example:
m_mdr = on
m_ddr = on
m_ydr = off

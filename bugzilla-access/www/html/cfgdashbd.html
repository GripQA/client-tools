Dashboard Section:

  'dashpath' – value is the filesystem pathname of the directory to receive the HTML dashboard page.  The pathname should be relative to the current working directory.  Specify nodash as the value (or comment out/remove the 'dashpath' option value pair completely) to skip dashboard generation.

As a security precaution, the HTML dashboard's filesystem location is fixed when the application is running as a service.   However, the nodash value will be effective.

Example 1:
dashpath = ../dashres/
         HTML Dashboard will be placed in the specified directory

Example 2:
dashpath = nodash
         HTML Dashboard will not be generated / displayed
